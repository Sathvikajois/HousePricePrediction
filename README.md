# House Price Prediction

This project predicts house prices using machine learning techniques.

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- Jupyter Notebook

## Project Workflow
1. Data Loading and Exploration
2. Data Cleaning and Preprocessing
3. Feature Encoding
4. Model Building
   - Linear Regression
   - Random Forest Regressor
5. Model Evaluation
6. Data Visualization
7. Insights and Recommendations

## Results
- Linear Regression R² Score: 0.653
- Random Forest R² Score: 0.612

Linear Regression performed better on this dataset.
